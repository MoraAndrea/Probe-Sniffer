| Message                       | Destination Address       | Port      | ProtocolType  |Header (1 byte)    | Payload  ("{lenght}" : 2byte)                     | Payload Len   | Examples                                  |   
|---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|   
| OK_MSG                        | ip_esp_device             | 45445     | TCP           |"200"              |                                                   | 0             |                                           |   
| SERVER_ADVERTISEMENT          | broadcast                 | 45445     | UDP           |"201"              |"{length}{IP}"                                     | variable      | (e.g. “192.168.1.1”)                      |   
| TIMESTAMP                     | ip_esp_device             | 45445     | TCP           |"202"              |"{AAAA}:{MM}:{GG}:{hh}:{mm}:{ss}"                  | 19            | (e.g. “2018:05:06:08:32:17”)              |   
| RESET                         | ip_esp_device             | 45445     | TCP           |"203"              |                                                   | 0             |                                           |
